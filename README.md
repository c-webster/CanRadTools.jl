# CanRadTools

[![Build Status](https://github.com/c-webster/CanRadTools.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/c-webster/CanRadTools.jl/actions/workflows/CI.yml?query=branch%3Amain)


## Description

CanRadTools is a collection of functions that are useful for preparing to run CanRad or for working with CanRad outputs. 

The package is currently under development and will be added to as new tools are developed.


## Installation

```
Pkg.add(url="https://github.com/c-webster/CanRadTools.jl")
```
